## 配置

在项目根目录创建 `.env` 文件，参考 `.env.example`：

```env
APP_PORT=3000
APP_ENV=development
DATABASE_URL=your_database_url
```

| 配置项         | 默认值        | 说明         |
| -------------- | ------------- | ------------ |
| `APP_PORT`     | `3000`        | 服务监听端口 |
| `APP_ENV`      | `development` | 运行环境     |
| `DATABASE_URL` | -             | 数据库连接串 |
